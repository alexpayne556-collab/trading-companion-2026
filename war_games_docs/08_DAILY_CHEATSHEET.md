## PRE-MARKET (7:00 AM - 9:30 AM ET)

### 7:00 AM - L1 RADAR (3 min)
```
□ Check SPY/QQQ/IWM pre-market
  → All green = Aggressive day
  → Mixed = Normal day  
  → All red = Defensive day

□ Check VIX level
  → <18 = Calm, be aggressive
  → 18-22 = Normal
  → >22 = Fearful, be careful

□ Scan sector heatmaps
  → Which sectors are GREEN?
  → Note the 2-3 hottest sectors

□ Check news for catalysts
  → Any contract awards?
  → Any earnings?
  → Any sector news?
```

### 7:30 AM - L2 FILTER (5 min)
```
□ Load hot sector watchlist
□ Sort by Vol % Avg (descending)
□ Click through top 5-7 names:
  → Vol % Avg > 150%? 
  → Daily above 21 EMA?
  → 5-min above VWAP?
□ Write down 2-3 REAL candidates
```

### 8:00 AM - L3 SNIPER (10 min per candidate)
```
For each candidate:
□ Mark support level (draw line)
□ Mark resistance level (draw line)
□ Determine entry price
□ Determine stop price
□ Determine target price
□ Calculate R:R (must be 2:1+)
□ Calculate position size

ENTRY: $_______ STOP: $_______ TARGET: $_______
SHARES: _______ R:R: _____:1
```

### 9:25 AM - FINAL CHECK
```
□ Order pre-filled in L4 (not submitted)
□ Market still supports my trade (check L1)
□ I know exactly what I'm doing
□ WAIT for market open
```

## MARKET OPEN (9:30 AM - 11:00 AM ET)

### 9:30 - 9:45 AM - WATCH ONLY
```
□ Stay in L1 or L2
□ DO NOT TRADE first 15 minutes
□ Let opening chaos settle
□ Watch if candidates hold levels
```

### 9:45 - 10:30 AM - EXECUTION WINDOW
```
When price approaches your entry:
□ Switch to L4
□ Run 30-second checklist:
  → Level 2: Bids holding?
  → Time & Sales: Green prints?
  → Chart: Above VWAP?
□ If 2+ YES → Execute
□ IMMEDIATELY set stop loss
□ Move to L5
```

### 10:30 AM - 11:00 AM
```
□ Manage positions in L5
□ Move stop to breakeven at +1R
□ Check L1 for other opportunities
```

## POSITION MANAGEMENT CHEAT SHEET

```
AT +1R (profit = risk):
→ Move stop to BREAKEVEN

AT +1.5R to +2R:
→ Sell 50% of position
→ Keep stop at breakeven

AT TARGET (+2R to +3R):
→ Sell another 25%
→ Trail stop on rest

STOP HIT:
→ OUT. No questions.
→ Don't re-enter immediately
```
