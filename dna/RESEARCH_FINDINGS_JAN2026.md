## THE SCANNER CRITERIA

### BUY SIGNAL (All must be true)

```python
def is_buy_signal(row):
    return (
        row['rsi'] < 40 and                    # Oversold
        row['dist_from_20d_high'] < -0.15 and  # Down 15%+ from high
        row['vol_ratio'] < 1.5 and             # Normal volume
        row['ret_5d_past'] < 0                 # Coming from weakness
    )
```

### SELL SIGNAL (Any triggers exit)
```python
def is_sell_signal(row):
    return (
        row['rsi'] > 70 or                     # Overbought
        row['vol_ratio'] > 2.0 or              # Volume exhaustion
        row['dist_from_20d_high'] > -0.05      # Near highs
    )
```

### OPTIMAL ENTRY SCORE
```python
def entry_score(row):
    score = 0
    if row['rsi'] < 40: score += 1
    if row['rsi'] < 30: score += 1
    if row['dist_from_20d_high'] < -0.10: score += 1
    if row['dist_from_20d_high'] < -0.20: score += 1
    if row['ret_5d_past'] < 0: score += 1
    if row['vol_ratio'] < 1.5: score += 1
    if row['is_monday'] or row['is_friday']: score += 1
    if row['volatility_10d'] < row['volatility_20d'] * 0.7: score += 1
    return score
```
