## MASTER PROMPT #2: SINGLE TICKER DEEP DIVE

Use this when you're ready to trade a specific ticker:

```
I'm preparing to trade [TICKER] and need a complete war game analysis.

ANALYZE:

**COMPANY FUNDAMENTALS**
- Current market cap, cash position, debt
- Revenue trend (last 4 quarters)
- Path to profitability (if not profitable)
- Key customers/contracts
- Competitive moat

**TECHNICAL POSITION**
- Current price vs 50-day and 200-day moving averages
- Distance from 52-week high/low
- Recent volume trends
- Key support levels (where has it bounced?)
- Key resistance levels (where has it failed?)

**CATALYST CALENDAR**
- Next earnings date
- Expected government contract decisions
- Regulatory milestones
- Product launches
- Conference presentations
- Insider lockup expirations

**NEWS ANALYSIS (Last 30 days)**
- Major announcements
- Analyst rating changes
- Institutional buying/selling (13F filings)
- Short interest changes
- Social media sentiment shift

**SCENARIO ANALYSIS FOR THIS TICKER**

Scenario 1: BULL CASE (Everything goes right)
- What happens?
- Price target?
- Probability?

Scenario 2: BASE CASE (Normal execution)
- What happens?
- Price target?
- Probability?

Scenario 3: BEAR CASE (Things go wrong)
- What could go wrong?
- Downside price target?
- Probability?

Scenario 4: DISASTER CASE (Black swan)
- What's the worst case?
- How low could it go?
- Probability?

**TRADE RECOMMENDATION**
- Entry zone (price range to buy)
- Stop loss level (where thesis breaks)
- Target 1 (1.5R)
- Target 2 (2R)
- Target 3 (3R if it really works)
- Position sizing recommendation (% of portfolio)
- Hold time expectation

**RISKS I MUST ACCEPT**
- List the top 5 risks if I take this trade
- What would make you NOT take this trade?
```

```
Build me a Q1 2026 catalyst calendar for these tickers:

DEFENSE: RCAT, KTOS, AVAV, PLTR, LHX

QUANTUM: IONQ, RGTI, QBTS, SEALSQ
SPACE: RKLB, ASTS, LUNR, BKSY
AI INFRA: VRT, MRVL, CRDO, ALAB
ROBOTICS: TER, ISRG, SYM
CYBER: CRWD, PANW, ZS
BIOTECH: RXRX, TWST, CRSP

FOR EACH TICKER, FIND:

1. **Earnings Dates** (confirmed or estimated)

2. **Government/Regulatory Events**
   - Contract award decisions
   - FDA approvals
   - NRC decisions
   - DoD budget votes
   - NASA milestones

3. **Company Events**
   - Product launches
   - Conference presentations
   - Investor days
   - Guidance updates

4. **Macro Events That Affect Them**
   - Federal Reserve meetings
   - Jobs reports (if relevant)
   - Geopolitical summits
   - Trade policy decisions

FORMAT AS:

**JANUARY 2026**
- [DATE] - [TICKER] - [EVENT] - [EXPECTED IMPACT: Bullish/Bearish/Neutral]

**FEBRUARY 2026**
- [DATE] - [TICKER] - [EVENT] - [EXPECTED IMPACT]

**MARCH 2026**
- [DATE] - [TICKER] - [EVENT] - [EXPECTED IMPACT]

Also flag:
- HIGH IMPACT events (could move stock 10%+)
- BINARY events (pass/fail outcomes)
- SECTOR-WIDE events (affect multiple tickers)
```
